# Mohamed Luqman — Video Editor Portfolio

Fully automated portfolio. Add a YouTube video to a playlist → it appears on the site within 24 hours. Delete a playlist → that category disappears automatically.

---

## How It Works

```
Your YouTube Playlists (one per category)
            ↓
  GitHub Actions (runs daily at midnight)
  fetch_videos.py → calls YouTube Data API
            ↓
     videos.json updated in repo
            ↓
  all-work.html reads videos.json dynamically
  → categories and videos render automatically
```

## One-Time Setup (15 minutes)

### Step 1 — Get a YouTube Data API Key
1. Go to https://console.cloud.google.com
2. Create a new project (or use existing)
3. Enable **YouTube Data API v3**
4. Go to **Credentials** → **Create Credentials** → **API Key**
5. Copy the key

### Step 2 — Find Your YouTube Channel ID
1. Go to your YouTube channel
2. URL will be like `youtube.com/channel/UCxxxxxxxxxxxxxx`
3. Copy the `UCxxxxxxxxxxxxxx` part
   (Or go to YouTube Studio → Settings → Channel → Advanced Settings → Channel ID)

### Step 3 — Add Secrets to GitHub
1. Open your GitHub repo
2. Go to **Settings** → **Secrets and variables** → **Actions**
3. Click **New repository secret** and add:
   - Name: `YOUTUBE_API_KEY` → Value: your API key from Step 1
   - Name: `YOUTUBE_CHANNEL_ID` → Value: your channel ID from Step 2

### Step 4 — Set Up YouTube Playlists
Create one playlist per category on your YouTube channel. Name them clearly:

| Playlist Name | What It Becomes |
|---|---|
| Tech | 💻 Tech category |
| Religious | 🕌 Religious category |
| Promotions | 📣 Promotions category |
| Long-Form | 🎬 Long-Form category (shows as 16:9) |
| Comedy | 😄 Comedy category |
| Motion GFX | 🎨 Motion GFX category |

Upload your YouTube Shorts/videos → add them to the right playlist. Done.

### Step 5 — Enable GitHub Pages
1. Go to repo **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / root
4. Save → your site will be at `https://yourusername.github.io/repo-name`

### Step 6 — Run the First Sync
1. Go to repo → **Actions** tab
2. Click **Update Videos from YouTube**
3. Click **Run workflow** → **Run workflow**
4. Wait ~30 seconds → check that `videos.json` is updated in your repo

## Daily Workflow (after setup)

| What you do | What happens |
|---|---|
| Upload a Short to YouTube | — |
| Add it to the right playlist | — |
| Wait until midnight UTC | GitHub Action runs automatically |
| | `videos.json` updates |
| | New video appears in All Work page |
| Delete a playlist on YouTube | Next day, that category vanishes from site |
| Create a new playlist | Next day, new category appears automatically |

### To trigger immediately (don't want to wait)
Go to **Actions** → **Update Videos from YouTube** → **Run workflow**

## What to Edit Manually

Only two sections in `index.html` need manual editing — marked clearly with `★`:

### Featured Reels (5 portrait videos)
Search for `★ REEL` in `index.html`. Replace the YouTube Short ID in the `src`:
```html
<!-- ★ REEL 1 — replace YouTube Short ID -->
<iframe src="https://www.youtube.com/embed/YOUR_SHORT_ID" allowfullscreen></iframe>
```

### Featured Long-Form (up to 5 landscape videos)
Search for `★ LONG-FORM` in `index.html`. Replace the video ID or Drive link:
```html
<!-- ★ LONG-FORM 1 -->
<iframe src="https://www.youtube.com/embed/YOUR_VIDEO_ID" allowfullscreen></iframe>
```

## File Structure

```
luqman-portfolio/
├── index.html              ← Main portfolio (hero, about, featured 5+5, contact)
├── all-work.html           ← Dynamic all-videos page (reads videos.json)
├── videos.json             ← Auto-generated by GitHub Actions (don't edit manually)
├── fetch_videos.py         ← Python script that fetches YouTube API
└── .github/
    └── workflows/
        └── update-videos.yml   ← GitHub Actions workflow (runs daily)
```

## Adding More Category Icons

Edit the `EMOJI_MAP` in `fetch_videos.py` and the `ICONS` object in `all-work.html`:

```python
# fetch_videos.py
EMOJI_MAP = {
    "gaming":    "🎮",   # add this
    "education": "🎓",   # add this
    # ...
}
```
